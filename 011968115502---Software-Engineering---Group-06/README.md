# Phone Book Management System

A desktop phone book application built with Python, Tkinter, and SQLite.

## Features

- Add, edit, delete, search, and view contacts.
- Store name, phone number, email, address, and optional group.
- Manage groups: create, rename, delete, and assign contacts to groups.
- Validate required fields, phone numbers, and email addresses before saving.
- Persist data in `phonebook.db` using SQLite.

## Requirements

- Python 3.10 or later.
- Tkinter, which is included with most standard Python installations.
